DevOps Environment

Project with specific CloudFormation templates, which publishes an environment for continuous integration of any project, for this it relies on 2 tools, Jenkins as the organizer of the integration and continuous deployment, and SonarQube for the code review of the projects. The templates contained here can be configured for the required load level, since the t2.micro instance type is used by default, but it can be changed to any type that is changed.
